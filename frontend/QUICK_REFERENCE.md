# 🚀 Quick Start Guide

## Installation & Running

```bash
# Install dependencies
cd frontend
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 📍 Navigation Map

```
🏠 Homepage (Dashboard)
├─ 📊 Dashboard - System overview & metrics
├─ 🔮 Predict - Make traffic predictions
├─ 📜 History - View all predictions
└─ 📈 Analytics - Deep insights & trends
```

---

## 🎯 Quick Tasks

### Make a Prediction
1. Click 🔮 **Predict**
2. Select date
3. Enter hour (0-23)
4. Input temperature (K)
5. Enter cloud cover (%)
6. Choose weather
7. Click "🚀 Predict Traffic"
✅ **Automatically saved to history**

### View History
1. Click 📜 **History**
2. (Optional) Filter by traffic level
3. (Optional) Search by weather
4. (Optional) Sort by date/volume/temp
5. View summary at bottom

### Analyze Data
1. Click 📈 **Analytics**
2. View KPI cards
3. Check hourly chart
4. See weather breakdown
5. Read insights

### Check System
1. Click 📊 **Dashboard**
2. Review KPIs
3. Check traffic pattern
4. See peak hours
5. Review system health


**What you get**:
```
Traffic Level (Low/Medium/High) + Volume (veh/h)
```

**Saved as**:
```
Timestamp + All above data
```
