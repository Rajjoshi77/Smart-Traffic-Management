# Support

If you need help running or extending TrafficIQ, use the following options:

- Check the `docs/DOCUMENTATION.md` and `docs/API_REFERENCE.md` for setup and API details.
- Open an issue on the GitHub repository with steps to reproduce and expected behavior.
- For urgent assistance, contact the maintainer: Raj Joshi (see repository contact info).

When filing issues, include environment details, commands run, and any logs or error messages.
