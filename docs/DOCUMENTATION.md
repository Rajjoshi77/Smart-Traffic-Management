# Project Documentation

## Overview

TrafficIQ is an AI-powered traffic analysis and prediction system built to support intelligent urban mobility solutions. This repository contains backend services, frontend UI, AI model training code, and big-data processing pipelines.

## Contents

- `frontend/` - React UI for interacting with TrafficIQ
- `backend/` - FastAPI backend and service integrations
- `ai_model/` - Model training utilities
- `big_data/`, `spark/` - Data generation and Spark analytics
- `Data/` - Raw and processed CSVs

## Getting Started

1. Create a Python virtual environment and install backend dependencies from `backend/requirements.txt`.
2. Install Node dependencies in `frontend/` and run the dev server.
3. Start the backend via `start_project.ps1` (Windows PowerShell) or run Uvicorn for the FastAPI app.
